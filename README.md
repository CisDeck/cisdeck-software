# CisDeck
## Streamdeck with custom Software and Hardware

### Hardware
The Hardware consists out of 2 main Components:
- A 4x4 Matrix Membrane Keypad
- An Arduino Nano

The packaging is made out of cardboard and tape due to missing materials.
